This is a console-based Loan Management System developed using C++. The project demonstrates Object-Oriented Programming (OOP) and Data Structures concepts by implementing Binary Search Trees and Linked Lists without using STL containers.

The system allows users to manage loans between people, track transactions, calculate fines, and view financial summaries.

✨ Features
👤 User Management

Add user

Update user

Delete user (soft delete using active flag)

Search user

List all active users

💰 Loan Operations

Borrow money between users

Return loans with fine calculation

Transfer loans between users

📊 Transaction Tracking

View account summary

View transactions between two users

System-wide financial summary

🏗️ Data Structures Used

Binary Search Tree (BST) → User management

Singly Linked List → Transactions storage

Dynamic Memory Allocation → Pointer-based memory handling

🛠️ Technologies Used

C++

Object-Oriented Programming

Console-based Interface
